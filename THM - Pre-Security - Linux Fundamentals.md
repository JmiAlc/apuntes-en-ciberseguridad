Hoy me he pegado con la terminal y he visto cómo interactuar con el sistema y herramientas de ciberseguridad.

### Herramientas de Reconocimiento Web
* **`dirb`**: Comando que se usa para rastrear y buscar páginas o "habitaciones" ocultas que los programadores se han dejado abiertas por despiste dentro de una web.
* **Resultados clave**: Cuando la pantalla se llena de texto, hay que buscar las líneas que empiezan con el signo **`+`**. Cada una es una página oculta que se ha encontrado y está accesible.
